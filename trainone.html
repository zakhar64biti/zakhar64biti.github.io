تمرین۱) فرض کنید فایل های ارایه شده، Assetها و محتوای وب‌سایت مسابقات جهانی Lyon 2024 هستند. به بهینه‌ترین روشی که فکر می‌کنید این محتوا (وب‌سایت مسابقات) را میزبانی کنید. دسترسی به مسیر /secure باید محدود شود. درواقع لازم است برای مسیر /secure یک مکانیزم هویت سنجی ساده تعبیه کنید (همانند شکل زیر) که با نام کاربری skilluser و رمز mypassword بتوان ورود کرد و محتوای مورد نظر را دید.

دیدن پورت های tcp
ss -nltp
روش پیاده سازی basic auth در nginx
sudo apt-get install apache2-utils
sudo yum install httpd-tools =====> in centOS
sudo htpasswd -c /etc/nginx/.htpasswd myuser ===> create user and password
server { listen 80; server_name _; # یا دامنه‌ات # روت عمومی (بدون لاگین) root /var/www/html; index index.html; # پوشه محافظت‌شده location /secure { # اگر فایل‌ها در /var/www/secure هستند، از alias استفاده کن: alias /var/www/secure/; index index.html; auth_basic "Restricted Area"; auth_basic_user_file /etc/nginx/.htpasswd; } }
sudo systemctl restart nginx. ====> restart nginx and test
